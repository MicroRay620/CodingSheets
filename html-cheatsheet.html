HTML Cheatsheet

Hypertext Markup Lanaguage or HTML is a language used to code websites (this website was even coded in HTML). HTML was invented in 1993. Files written in HTML use .html file indicator. This is one of the languages in the web 3 lanaguages. The other two are Cascading Stylesheets (or CSS) and JavaScript (which is also called <

NOTE: MAJORITY OF THE COMMAND WILL HAVE </> CONTAINED IN IT! Will say if a command is an exception to this rule. Every command (or tag) listed in this page will be written between <html> and </html>.

Key Terms

Before getting into the actual tags, let's get some terms defined that will be used.

  • Tags
    In coding, tags are what is put between < and >.
  • Attributes
    Attributes are additions to tags. This can be have a word tied to a css file or to a js file. It also can mean linking a word to a website url. These will be discussed in [insert section name].

Structure Tags

Structure tags are used to define areas which other tags can appear on the page.

  • <body> is used to define tags in the body portion of the page.
    This tag is the most common out of the two tags.
  • <foot> is used to define tags in the footer portion of the page. With it being uncommon to use, it can be used to have buttons in the bottom section of the page.

Text Tags

With the structure tags being defined, lets move to the tags that can be used to have text contained in them. This list will include some tags that can be used to add various changes to the text.

  • <h#> is used to make header text.
    The # can be replaced by any number between 1 and 6.
  • <p> is used to make standard text. The most common tag to find on any website for text.
    The p stands for paragraph.
    • <br> is used to add line break to your text (like in <p>).
      You won't have </br> used in your document.

      • THE NEXT TAGS ARE OFTEN DONE IN A CSS FILE

    • <b> is used to bold text.
      • <strong> is also used to bold text. (Honestly, if it has other uses to it, I do not know. Everytime I used <strong> I saw no difference between bold and strong.
    • <i> is used to italize text.
      • <em> is also used to italize text (it is truly used for empthasizing words, but I saw no diffference when I used this tag [in w3schools

Head Tags

These tags are used to link documents with each other and add more to your document. These use attributes to work.

All tags used here are between <head> and </head> portion of the document (which is often right below below the <html>).

Additional Key Terms

Before we get into the actual tags, lets define some key terms for this section.

  • Block-level
    Code that is using another file to define a command. An example would be like <span> for using css in an html document.
    NOTE: USING THIS NEEDS A CSS OR JS FILE TO BE LINKED TO! The ways to link those files will be explained in the <link> tag below.
  • In-line
    Code that is used for another language but is still in the same file (lke having css in an html file.)

Head Text Tags

These are tags used to input text that would appear in a spot on the page (like in the Web Page tab.

  • <title> is used to have text appear in the web browser tab.

File Tags

File tags are used to have css or JavaScript connected to/used in an html document.

  • <style> is used to have css code in an html file.
    (This is only used for in-line css code.)
  • <script> is used to have JavaScript in an html file.
    (This is used for in-line and block level JavaScript code.)
    • <script src="myjavascript.js"> </script> (NOTE: </script> is required to be used) is often used for block-level JavaScript.
      It can also have the additional attribute of type="text/javascript".
    • <link> is used to link a file to your html file.
      You wouldn't use </link> at all for this.
      • <link type="text/css" rel="stylesheet" href="css/style.css"> is used to link an external css file to your html file. You should have a seperate folder labeled css
        NOTE: in href="css/style.css", the style.css would be changed to the name of your css file.
    • <span> is used to identify what text in you html file that you want a css tag applied to.
      • <span> class="text"> is used to have text style that would be repeated in the html represented. Like bold or italics
