HTML Cheatsheet

Markdown is a language that is often used for notetaking. Markdown is a very simple language to learn, many websites and softwares use the Markdown language.
Markdown is identified by the .md file type.
You may know the Markdown language from places like Discord, Obsidian Note, and Notion.
