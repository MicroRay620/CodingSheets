Key Terms

Before getting into the actual tags, let's get some terms defined that will be used.

  • Tags
    In coding, tags are what is put between < and >.
  • Attributes
    Attributes are additions to tags. This can have a word tied to a css file or to a js file. It also can mean linking a word to a website url. These will be discussed in [insert section name]. This is explained more in Attributes.
  • Block-Level Code that is using another file to define a command. An example would be like <span> for using css in an html document.
    NOTE: USING THIS NEEDS A CSS OR JS FILE TO BE LINKED TO!
    The ways to link those files will be explained in the <link> tag below.
  • In-Line is code that is used for another language but is still in the same file (lke having CSS in an html file.)

Cheatsheet

The Cheat Sheet is here to help you find the tag or attribute that you're looking for.

Structure Tags

Structure tags are used to define areas which other tags can appear on the page.

  • <head> is used for define tags that aren't on the main portion (or body) of the page.
  • <body> is used to define tags in the body portion of the page.
    This tag is the most common out of the three tags.
  • <footer> is used to define tags in the footer portion of the page. With it being uncommon to use, it can be used to have buttons in the bottom section of the page.
    This tag is also able to be put under <body> as well as being below the body.

Head Tags

These are tags used to input text on things like the webpage, putCSS or onto the HTML file, or connect a CSS or JS file to the HTML file.

Head Text Tags

  • <title> is used to have text appear in the web browser tab.

File Tags

  • <style> is used to have css code in an html file.
    (This is only used for in-line CSS code.)
  • <script> is used to have JavaScript in an htmlfile.
    (This is used for in-line and block-level JavaScript code.)
    • <script src="myjavascript.js"> </script> (NOTE: </script> is required to be used) is often used for block-level JavaScript. It can also have the additional attribute of type="text/javascript".
    • <link> is used to link a file to your html file.
      You wouldn't use </link> at all for this.
      • <link type="text/css" rel="stylesheet" href="css/style.css"> is used to link an external css file to your html file. You should have a separate folder labeled css
        NOTE: in href="css/style.css", the style.css would be changed to the name of your CSS file.

Body Tags

Attributes

Attributes are things that can be added to tags. These can often be found with <div>, <span>, and <a>. Though attributes are not just limited to those tags, attributes can be on any tag, like <h1>, <p>, and other similar tags.
All of attributes will be on a tag and will be followed by " ".

  • id is used if a tag will be different and only used once in a file. This is often used for CSS.
    This only is only one time per file. it will be typed as id=" ", inside the quotation marks (" ") will have the name you want to identify the tag as.
    THIS TAG IS NEED IF YOU ARE DOING LINKING TO HEADINGS! The
  • class is similar to to id but can be used multiple times in a file. This will typed as class=" ", and just like id, class will have what you want to identify the class as between the quotation marks (" ").
  • href is used for linking files, pages, or headings.

Container Tags

Container tags are tags that you can use for connecting specific sections to CSS code. You will often see these tags for specific areas like sidebars, bolded words (unless it uses the HTML method, which is talked about in the Text Tags section), change in text color, font family (like Arial, Times New Roman, and Roboto), and many other changes in text. See CSS Cheatsheet for what these changes are.

  • <span> is used to identify what text in you html file that you want a css tag applied to.
    • <span class="text"> is used to have text style that would be repeated in the html represented. Like bold or italics
  • <div> is used for major and/or broad changes like layout, grouping sections, and scripting.
    Scripting with <div> is often done for dynamic elements, which often uses JavaScript

If you are just starting out with using HTML, I would recommend heading to w3schools or taking a web programming course at your school (if it has it available).

Text Tags

With the structure tags being defined, lets move to the tags that can be used to have text contained in them. This list will include some tags that can be used to add various changes to the text.

  • <h[#]> is used to make header text.
    The heading font sizes in the sub-table below (h1 to h6) are listed with their fonts on a 1920 pixel by 1080 pixel (1080p or a Full HD) computer monitor).
    The px next to the font sizes stands for pixel(s).
    • <h1> is used to make a Heading 1. This heading is 32px big.
    • <h2> is used to make Heading 2. This heading is 24px big.
  • <p> is used to make standard text. The most common tag to find on any website for text.
    The p stands for paragraph.
  • <br> is used to add line break to your text (like in <p>). You won't have </br> used your document.

    • THE NEXT TAGS ARE CAN ALSO BE DONE IN A CSS FILE

  • <b> is used to bold text.
    • <strong> is also used to bold text.
  • <i> is used to italize text.
    • <em> is also used to italize text
  • <a> is used for linking a word to a website, file, and/or heading (you can only combine file and heading).
    Though this does require you to use have it typed as <a href="[url, file path, and/or heading]">.
    • For linking to headings you will have to have the heading with <a href="#[heading name]">

List Tags

Lists are used to organize specific section on a webpage.

List Type tags

  • <ul> is used to make a bullet-point list.
  • <ol> is used to make a number list.
Point Tags

Pointer tags are used for the various points in a list, the points being the number of bullet.

  • <li> is used to make the bullets/numbers in a list.
    This tag will be between either the <ul> and </ul> or <ol> and </ol>.
