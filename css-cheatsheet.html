CSS Cheatsheet

Cascading Stylesheets or css is a programming language developted in 1994. CSS is commonly used to style an HTML document. Files written in css use the .css file indicator.

As stated in the html cheatsheet, the way you connect a css file to an html file is by doing
<link type="text/css rel="stylesheet" href="style.css">, with style.css being the placeholder for your css file.

All tags in html can contain any attribute listed.

Text Change Tags

Using CSS, it can allow you to change various aspects of the text. The list below will show you tags that affect the text. These changes are often down by using the tag indicator (the character(s) be tweeen the < > in the html document.
These tags can be done by using span or div. This, again, is talked about in the html cheatsheet

  • Color is used to change the color of the text.
    Can use common color name or its hexcode . (I recommend using the hexcode over the common name)
  • font-family is used to change what font your text is in. (Like having it be in arial)
  • font-size is used to change the size of the text.
    This does require it to be listed as font-size: #px; in the style portion of the page/in the css file.
